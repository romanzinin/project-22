Программы выдают задержку при выполнении n^2 заданных операций (т.е. n тестов по n операций в каждом); runs - в секундах, runms - в миллисекундах, при подсчёте времени используются разные библиотеки.
В файл input.txt неодходимо через пробем ввести сначала частоту процессора в МГц, а затем число n (рекомендуется использовать не менее 1000-5000).
Операцию можно задать на строчке 39 (начальное содержание - ix += iy;).
Работоспособность может зависеть от компилятора, в моём всё работает нормально, но в другом уже выдавались отрицательные значения.
